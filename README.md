# avengers-hub

AI 复仇者团队中枢 - 任务追踪、知识库、协作中心  
同时也是从前端工程师转型 AI Agent 工程师的 3 个月系统化路线图。

---

## 这个仓库是什么

记录"复仇者 AI 团队"的技术实践，以及从前端工程师转向 AI Agent 工程师的完整规划和执行过程。

服务器上已部署 OpenClaw + 6 个 AI Agent（friday / captain / black_widow / doctor_strange / iron_man / vision），这个仓库是规划后续学习路线、沉淀学习成果的基地。
